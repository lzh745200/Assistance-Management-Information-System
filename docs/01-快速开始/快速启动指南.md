# 快速启动指南

## 方式一：开发模式（前后端分离，推荐开发时使用）

### 使用现有脚本（最简单）

**双击运行：`一键启动.bat`**

该脚本会：
1. 检查 Python 和 Node.js
2. 安装依赖
3. 初始化数据库
4. 启动后端（新窗口）
5. 启动前端（新窗口）

**访问地址：**
- 前端：http://localhost:5173
- 后端 API：http://localhost:8000
- API 文档：http://localhost:8000/docs

**停止服务：** 关闭后端和前端服务窗口

---

## 方式二：一体化模式（单进程，推荐生产/测试）

### 首次使用需要构建前端

1. **构建前端**：
   ```bat
   cd frontend
   set VITE_API_BASE_URL=
   npm run build
   ```

2. **拷贝到后端**：
   ```bat
   xcopy /E /I /Y frontend\dist\* backend\static\
   ```

3. **启动后端**：
   ```bat
   cd backend
   python -m uvicorn app.main:app --host 0.0.0.0 --port 8000
   ```

**访问地址：** http://localhost:8000（前端与接口同源）

---

## 方式三：使用打包后的安装包

如果您已经打包了系统：

1. **解压安装包**到目标目录
2. **运行安装向导**：双击 `安装向导.bat`
3. **启动系统**：双击 `运行系统.bat`
4. **访问**：http://localhost:8000

## 常见问题

### 端口被占用

**检查端口：**
```bat
netstat -ano | findstr :8000
```

**关闭占用进程：**
```bat
taskkill /PID <PID> /F
```

### Python 未找到

- 确保已安装 Python 3.8+
- 检查 PATH 环境变量
- 重新安装 Python 并勾选「Add to PATH」

### 浏览器无法访问

- 检查防火墙是否放行 8000 端口
- 尝试 http://127.0.0.1:8000
- 查看后端日志：`backend/logs/app.log`

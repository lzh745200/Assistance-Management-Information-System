# 军队乡村振兴帮扶管理系统 - 使用指南

## 📋 系统简介

**系统名称：** 军队全面推进乡村振兴工作管理系统
**版本：** 1.0.3
**交付形式：** Electron 桌面应用（离线单机版）
**版权所有：** © 2025 梁正辉

本系统是一套完整的乡村振兴帮扶管理平台，以 Electron 桌面应用形式交付，内嵌 FastAPI 后端服务和 SQLite 数据库，无需网络即可运行。用于管理帮扶村、帮扶项目、经费、数据等相关信息。

## 🚀 快速开始

### 1. 系统要求

#### 运行环境（安装包用户）
- **操作系统：** Windows 10/11、银河麒麟 V10（AMD64/ARM64）
- 双击安装包即可使用，无需额外安装 Python 或 Node.js

#### 开发环境
- **Python：** 3.11 或更高版本
- **Node.js：** 18 或更高版本
- **浏览器：** Chrome、Edge、Firefox 等现代浏览器

#### 硬件要求
- **内存：** 至少 4GB RAM（推荐 8GB）
- **磁盘空间：** 至少 2GB 可用空间
- **处理器：** 双核及以上

### 2. 启动系统

#### 方式一：使用启动脚本（推荐）

**Windows用户：**
1. 双击运行 `START_SYSTEM.bat`
2. 选择启动方式：
   - 选项1：仅启动后端服务
   - 选项2：仅启动前端服务
   - 选项3：同时启动前后端（推荐）
3. 等待服务启动完成
4. 系统会自动在浏览器中打开

#### 方式二：手动启动

**启动后端服务：**
```bash
# 进入后端目录
cd backend

# 创建虚拟环境（首次运行）
python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# 启动后端服务
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

**启动前端服务：**
```bash
# 进入前端目录
cd frontend

# 安装依赖（首次运行）
npm install

# 启动前端服务
npm run dev
```

### 3. 访问系统

服务启动成功后，可以通过以下地址访问系统：

- **前端地址：** http://localhost:5173
- **后端地址：** http://localhost:8000
- **API文档：** http://localhost:8000/docs

### 4. 停止系统

**Windows用户：**
- 双击运行 `STOP_SYSTEM.bat` 停止所有服务

**手动停止：**
- 在运行服务的命令行窗口中按 `Ctrl + C`

## 📖 功能使用

### 1. 帮扶村管理

#### 访问路径
侧边栏 > 帮扶村管理

#### 主要功能
- **查看帮扶村列表：** 显示所有帮扶村的基本信息
- **新增帮扶村：** 点击"新增帮扶村"按钮添加新记录
- **编辑帮扶村：** 点击"编辑"按钮修改帮扶村信息
- **查看详情：** 点击"查看"按钮查看帮扶村详细信息
- **管理年度数据：** 点击"年度数据"按钮管理该帮扶村的年度数据
- **删除帮扶村：** 点击"删除"按钮删除帮扶村记录

#### 年度数据管理
帮扶村的年度数据包括11个板块：
1. 人口数据
2. 收入数据
3. 力量投入
4. 产业帮扶
5. 基础设施
6. 党建帮扶
7. 医疗帮扶
8. 消费帮扶
9. 就业帮扶
10. 教育帮扶

### 2. 数据导入

#### 访问路径
侧边栏 > 数据管理 > 数据导入

#### 导入步骤
1. **下载模板：** 选择要导入的数据类型，下载对应的Excel模板
2. **填写数据：** 按照模板格式填写数据
3. **上传文件：** 上传填写好的Excel文件
4. **查看结果：** 查看导入结果，包括成功和失败的记录

#### 注意事项
- 不要修改模板的表头和结构
- 所有必填项都必须填写完整
- 日期格式请使用YYYY-MM-DD
- 金额字段请保留两位小数
- 下拉选项字段请严格按照选项列表填写

### 3. 数据导出

#### 访问路径
帮扶村管理页面 > 导出数据按钮

#### 导出类型
- **基本信息导出：** 导出所有帮扶村的基本信息
- **综合数据导出：** 导出指定年份的所有帮扶村综合数据，包含11个工作表

### 4. 经费管理

#### 访问路径
侧边栏 > 经费管理

#### 主要功能
- **查看经费统计：** 查看总预算、部队投入、地方投入、已使用、结余、使用率
- **查看分类统计：** 查看各帮扶类型的经费使用情况
- **筛选查询：** 按帮扶村、经费类型、帮扶类型、年份等条件筛选
- **新增记录：** 添加新的经费记录
- **编辑记录：** 修改已有经费记录
- **删除记录：** 删除不需要的经费记录

### 5. 数据备份

#### 方式一：普通用户备份
1. 进入侧边栏 > 数据管理 > 数据备份
2. 点击"创建备份"按钮
3. 可选填写备份描述和加密密码（设置密码后备份文件将使用 AES-256 加密）
4. 等待备份完成
5. 可在列表中查看历史备份记录

> 注：普通用户可创建和查看备份，恢复和删除备份需联系管理员。

#### 方式二：管理员备份管理
1. 进入侧边栏 > 数据管理 > 数据备份管理（仅管理员）
2. 查看所有备份文件，支持预览、验证、恢复、删除
3. 支持加密备份（AES-256）
4. 支持清理过期备份

#### 定时自动备份（管理员）
1. 在备份管理页面的"定时自动备份"卡片中启用开关
2. 设置备份间隔（每6/12/24小时、每3天、每周）
3. 设置保留份数和每日备份时间
4. 点击"保存配置"启动定时任务

### 6. 组织管理

#### 访问路径
侧边栏 > 组织管理（仅管理员可见）

#### 主要功能
- 查看组织结构
- 新增组织
- 编辑组织信息
- 删除组织

### 7. 数据管理

#### 访问路径
侧边栏 > 数据管理

#### 主要功能
- 数据导入
- 数据导出
- 数据统计
- 数据分析

### 8. 地图可视化

#### 访问路径
侧边栏 > 地图可视化

#### 主要功能
- **地图显示：** 支持离线 Leaflet 瓦片底图和 ECharts 矢量地图双模式
- **帮扶村标注：** 显示所有帮扶村地理位置，支持按单位筛选
- **学校标注：** 显示帮扶学校位置和帮扶状态
- **距离计算：** 自动计算到军分区的直线距离和山区公路预估车程（弯道系数1.4，平均时速45km/h）
- **县距离汇总：** 显示到黔南州12县市的距离概览
- **坐标编辑（管理员）：** 在村庄/学校详情弹窗中可手动输入经纬度更新坐标

## 🔧 常见问题

### Q1: 系统无法启动怎么办？

**A:** 请检查以下几点：
1. Python和Node.js是否正确安装
2. 端口8000和5173是否被占用
3. 依赖是否完整安装
4. 查看错误日志了解具体问题

### Q2: 数据导入失败怎么办？

**A:** 请检查以下几点：
1. Excel文件格式是否符合模板要求
2. 数据是否完整填写
3. 字段格式是否正确
4. 查看错误日志了解具体错误信息

### Q3: 如何重置管理员密码？

**A:** 运行以下命令：
```bash
python reset_admin.py
```

### Q4: 系统支持哪些浏览器？

**A:** 推荐使用以下浏览器：
- Chrome（推荐）
- Edge
- Firefox
- Safari

### Q5: 数据存储在哪里？

**A:** 数据存储在以下位置：
- **数据库：** `backend/data/` 目录
- **上传文件：** `backend/uploads/` 目录
- **备份文件：** `backend/app/backups/` 目录

### Q6: 如何备份数据？

**A:** 有两种方式：
1. 在主页点击"一键备份"按钮
2. 进入"系统管理" > "备份管理"手动备份

### Q7: 系统支持多用户吗？

**A:** 是的，系统支持多用户，不同用户可以有不同的权限级别。

### Q8: 如何导出数据？

**A:** 在帮扶村管理页面点击"导出数据"按钮，选择导出类型和年份即可。

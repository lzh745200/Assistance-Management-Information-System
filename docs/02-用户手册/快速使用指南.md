# 快速使用指南

## 🚀 快速开始

### 方式一：直接打开（推荐）

1. 找到项目根目录下的 `index.html` 或 `home.html` 文件
2. 右键点击文件
3. 选择"打开方式" → 选择您的浏览器（Chrome、Edge、Firefox等）
4. 系统将在浏览器中打开

### 方式二：使用启动脚本

1. 双击 `启动单机版系统.bat` 文件
2. 系统会自动使用默认浏览器打开

## 💾 数据管理

### 数据存储

- 所有数据保存在浏览器的 localStorage 中
- 数据会自动保存，无需手动操作
- **重要**: 请勿清除浏览器缓存，否则数据会丢失

### 数据备份

**定期备份数据:**
1. 进入"系统管理" > "数据备份"
2. 点击"备份数据"
3. 系统会创建备份点

**导出数据:**
1. 点击"导出数据"按钮
2. 数据会以 JSON 文件形式下载
3. 保存文件到安全位置

**恢复数据:**
1. 在备份列表中选择备份点
2. 点击"恢复"按钮
3. 确认恢复操作
4. 数据将被还原

## 🔒 安全建议

### 密码安全

1. 首次登录后，建议修改默认密码
2. 密码应包含字母、数字，长度不少于6位
3. 不要与他人分享管理员密码

### 权限管理

1. 谨慎分配管理员权限
2. 定期检查用户权限
3. 及时禁用不再使用的账号

### 数据安全

1. 定期备份数据（建议每周备份）
2. 将备份文件保存到多个位置
3. 重要数据导出后妥善保管

## ⚠️ 注意事项

1. **浏览器兼容性**
   - 推荐使用: Chrome、Edge、Firefox
   - 不推荐使用: IE浏览器

2. **数据安全**
   - 不要清除浏览器缓存
   - 定期执行数据备份
   - 导出重要数据

3. **性能优化**
   - 数据量较大时，操作可能变慢
   - 建议定期清理历史日志
   - 删除不需要的文件

4. **离线使用**
   - 系统完全支持离线运行
   - 无需网络连接
   - 所有功能均可正常使用

## 🆘 常见问题

### Q: 为什么打不开系统？

A: 请确保使用现代浏览器打开 HTML 文件，右键选择"打开方式"选择浏览器。

### Q: 数据丢失怎么办？

A: 如果有备份，可以在"系统管理" > "数据备份"中恢复。如果没有备份，数据无法恢复。

### Q: 忘记密码怎么办？

A: 管理员可以在"系统管理" > "用户管理"中重置用户密码。如果忘记管理员密码，需要清除浏览器数据重新初始化。

### Q: 如何清除所有数据重新开始？

A: 

1. 打开浏览器开发者工具（F12）
2. 进入 Application/应用程序 标签
3. 选择 Local Storage
4. 删除所有 `military_village_` 开头的项
5. 刷新页面

### Q: 可以在多台电脑上使用吗？

A: 可以，但每台电脑的数据是独立的。如果需要同步数据，可以使用"导出数据"和"导入数据"功能。

### Q: 系统支持多少数据量？

A: localStorage 通常限制为 5-10MB，足够存储数千条记录。如果数据量超大，建议使用完整版（需要后端支持）。

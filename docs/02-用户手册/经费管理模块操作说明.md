# 经费管理模块操作说明

## 一、模块概述

经费管理模块是军队乡村振兴管理系统的核心模块之一，提供经费全生命周期管理功能，包括经费申请、审批、拨付、使用、核销、审计等完整工作流程，以及多维度统计分析、数据导出和文件管理功能。

### 功能架构

```
经费管理
├── 经费列表（EnhancedList）    — 经费记录总览、搜索、快捷操作
├── 经费详情（Detail）          — 完整信息展示、工作流操作、附件管理
├── 统计分析（Analysis）        — 多维度统计图表、数据分析
├── 经费报表（Report）          — 报表生成、打印、导出
└── 数据导出（Export）          — 多格式导出（CSV/Excel/PDF）
```

## 二、经费列表

### 2.1 访问路径
侧边栏 → 经费管理 → 经费列表

### 2.2 功能说明

| 功能 | 说明 |
|------|------|
| 搜索 | 支持按名称关键字搜索经费记录 |
| 类型筛选 | 项目经费、运营经费、教育帮扶、基础设施、应急经费、其他 |
| 状态筛选 | 待审批、已批准、已拨付、使用中、已完成、已审计、已驳回 |
| 新增 | 点击"新增"按钮创建新的经费记录 |
| 快捷审批 | 对待审批状态的记录，可直接点击"审批"按钮快速审批 |
| 快捷拨付 | 对已批准状态的记录，可直接点击"拨付"按钮快速拨付 |
| 导出 | 点击"导出"按钮将当前列表导出为 CSV 文件 |
| 查看详情 | 点击记录行可跳转至详情页 |

### 2.3 操作步骤

**新增经费记录：**
1. 点击列表页右上角的"新增"按钮
2. 填写表单：名称、类型、来源、金额、申请人等
3. 点击"确认"保存，新记录状态为"待审批"

## 三、经费详情与工作流

### 3.1 访问路径
经费列表 → 点击某条记录 → 进入详情页

### 3.2 基本信息展示
详情页展示经费的完整信息，包括：
- **基本信息**：编号、名称、类型、来源、申请人、申请日期
- **金额信息**：申请金额、批准金额、拨付金额、已使用金额、剩余金额
- **审批信息**：审批人、审批日期、拨付日期、拨付方式
- **审计信息**：审计日期、审计结果、审计意见
- **备注说明**：用途说明、备注

### 3.3 工作流操作

经费状态流转遵循以下状态机：

```
待审批(pending) → 已批准(approved) → 已拨付(allocated) → 使用中(in_use) → 已完成(completed) → 已审计(audited)
                ↘ 已驳回(rejected)
```

**各环节操作说明：**

1. **审批** — 管理员对"待审批"状态的经费进行审批
   - 填写审批意见（可选）
   - 点击"批准"或"驳回"

2. **拨付** — 对"已批准"状态的经费进行拨付
   - 填写拨付金额（必填）
   - 选择拨付方式（银行转账/现金/支票等）
   - 填写备注（可选）

3. **开始使用** — 将"已拨付"状态的经费标记为使用中

4. **完成** — 将"使用中"的经费标记为已完成

5. **审计** — 对"已完成"的经费进行审计
   - 选择审计结果（通过/未通过）
   - 填写审计意见

> **权限说明**：工作流操作按钮仅对管理员角色可见。普通用户只能查看经费详情。

### 3.4 附件管理

在详情页的"相关附件"区域，可以进行文件管理：

**上传附件：**
1. 点击"上传附件"按钮
2. 选择附件分类：合同、发票、收据、报告、其他
3. 选择文件（支持 pdf/doc/docx/xls/xlsx/jpg/jpeg/png/gif/bmp/txt/zip/rar 格式）
4. 系统自动上传并关联到当前经费记录

**预览附件：**
- 图片文件（jpg/png/gif/bmp）：点击预览按钮，弹窗内查看大图
- PDF 文件：点击预览按钮，新窗口中打开查看
- 其他格式：使用下载方式查看

**下载/删除附件：**
- 点击下载按钮可下载附件到本地
- 点击删除按钮（确认后）删除附件

## 四、统计分析

### 4.1 访问路径
侧边栏 → 经费管理 → 统计分析

### 4.2 维度切换
页面顶部提供四个分析维度的切换：

| 维度 | 说明 |
|------|------|
| 时间维度 | 按月度/季度/年度聚合统计，可选择时间粒度 |
| 类型维度 | 按经费类型（项目/运营/教育/基础设施/应急/其他）分组 |
| 来源维度 | 按经费来源（军队/政府/捐赠/企业/其他）分组 |
| 状态维度 | 按经费状态（待审批/已批准/已拨付/使用中等）分组 |

### 4.3 图表说明
- **分布饼图**：展示各分组的金额占比
- **利用率柱状图**：展示各分组的经费利用率（已使用/总金额）
- **年度对比图**：展示不同年份的经费使用对比趋势
- **统计明细表格**：列出每个分组的记录数、总金额、已拨付、已使用、利用率

### 4.4 导出统计
点击"导出统计"按钮，可将当前统计明细导出为 CSV 文件。

## 五、经费报表

### 5.1 访问路径
侧边栏 → 经费管理 → 经费报表

### 5.2 报表生成
1. 选择报表类型：月度报表 / 季度报表 / 年度报表
2. 设置时间范围（可选）
3. 选择经费类型筛选（可选）
4. 点击"生成报表"

### 5.3 报表内容
生成后的报表包含：
- **数据表格**：经费记录明细，包含日期、项目名称、类型、金额、使用率、状态等
- **统计汇总**：总经费、已使用、剩余、平均使用率四个指标
- **类型分布图**：环形图展示各类型经费占比
- **使用趋势图**：折线图展示总金额与已使用金额的变化趋势

### 5.4 报表导出与打印
- 点击"导出"按钮将报表数据导出为 CSV 文件
- 点击"打印"按钮调用浏览器打印功能

## 六、数据导出

### 6.1 访问路径
侧边栏 → 经费管理 → 数据导出

### 6.2 导出配置

**导出格式选择：**
- **CSV 格式**：调用后端导出接口生成，包含 BOM 头可直接用 Excel 打开，支持中文
- **Excel 格式**：前端生成（基于 CSV），适合日常使用
- **PDF 格式**：生成打印预览页面，支持在线查看和打印

**筛选条件：**
- 经费类型：可多选（项目经费、运营经费、教育帮扶、基础设施、应急经费、其他）
- 状态筛选：可多选（待审批、已批准、已拨付、使用中、已完成、已审计、已驳回）
- 包含字段：可自定义选择需要导出的字段

### 6.3 数据预览
导出前，页面下方会显示数据预览表格，展示将要导出的数据内容和预计文件大小。切换筛选条件会自动刷新预览。

### 6.4 操作步骤
1. 选择导出格式
2. 设置筛选条件
3. 确认预览数据无误
4. 点击"开始导出"
5. 文件自动下载到本地

## 九、常见问题

### Q1：为什么我看不到工作流操作按钮？
工作流操作（审批、拨付、审计等）仅对管理员角色可见。请确认您的账户具有管理员权限。

### Q2：上传附件时提示"不允许的文件类型"？
系统仅支持以下格式文件：pdf、doc、docx、xls、xlsx、jpg、jpeg、png、gif、bmp、txt、zip、rar。请检查文件扩展名是否在支持列表内。

### Q3：导出的 CSV 文件用 Excel 打开是乱码？
后端导出的 CSV 文件已包含 BOM 头，应能正确显示中文。如仍有乱码，请尝试：
1. 用 Excel → 数据 → 从文本/CSV 导入，选择 UTF-8 编码
2. 或使用 WPS 打开

### Q4：统计数据与列表数据不一致？
统计分析页面的数据来源于后端聚合查询，与列表的分页数据展示可能存在时间差异。点击"查询"按钮可刷新最新数据。

### Q5：系统为单机离线版，数据如何备份？
数据存储在本地 SQLite 数据库中，数据库文件路径为 `backend/data/` 目录下。建议定期将该目录复制到外部存储介质进行备份。

## 十、技术参数

- **后端框架**：FastAPI + SQLAlchemy + SQLite
- **前端框架**：Vue 3 + Element Plus + ECharts
- **桌面运行环境**：Electron
- **支持操作系统**：Windows 10/11、银河麒麟 V10
- **数据库**：SQLite（本地存储，无需网络连接）

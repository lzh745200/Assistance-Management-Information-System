# 军队乡村振兴管理系统 — 系统概览

## 系统定位

军队全面推进乡村振兴工作管理系统（简称 MRRMS）是一套面向军队参与乡村振兴帮扶工作的 **离线单机版桌面管理平台**。系统以 Electron 桌面应用形式交付，内嵌 FastAPI 后端与 SQLite 数据库，无需网络即可完成帮扶村管理、项目跟踪、资金监管、数据统计分析等全流程业务。

- **版本**：1.0.3
- **运行环境**：Windows 10/11、银河麒麟 V10（AMD64/ARM64）

## 技术架构简图

```
┌──────────────────────────────────────────────────┐
│              Electron 28 桌面壳                    │
│  ┌──────────────────┐  ┌──────────────────────┐  │
│  │ Vue 3 + TS 前端  │  │ FastAPI + Python 后端 │  │
│  │ Element Plus     │←→│ SQLAlchemy + SQLite   │  │
│  │ ECharts/Leaflet  │  │ JWT + bcrypt 认证     │  │
│  └──────────────────┘  └──────────────────────┘  │
│                    ↕ 本地磁盘                      │
│          SQLite 数据库 / 文件存储 / diskcache      │
└──────────────────────────────────────────────────┘
```

## 核心功能模块

| 模块 | 功能概述 |
|------|----------|
| 帮扶村管理 | 17 个数据板块，覆盖人口/收入/产业/基础设施/党建/医疗/教育等 |
| 项目管理 | 帮扶项目全生命周期管理与进度跟踪 |
| 经费管理 | 经费审批→拨付→使用→审计全工作流，附件管理 |
| 数据分析 | ECharts 多维度可视化、分析仪表盘、报表订阅 |
| 地图可视化 | Leaflet/ECharts 双模式，帮扶村/学校标注，车程估算 |
| 数据管理 | 12 种 Excel 模板、4 步导入向导、11 工作表综合导出 |
| 备份管理 | ZIP 打包、AES-256 加密、定时自动备份、过期清理 |
| 系统管理 | 用户/角色/组织管理、RBAC 权限、审计日志 |

# 帮扶管理信息系统 - 系统架构说明

## 📐 系统架构概览

本系统采用 **Electron 桌面壳 + 前后端分离** 的架构设计，以离线单机版形式交付。

```text

│              Electron 28 桌面壳 (Desktop Shell)          │

│  └───────────────────────────────────────────────┘  │

                         │
          ┌────────────┼────────────┐

│  Vue 3 + TypeScript     │ │  Python 3.11 + FastAPI    │
│  Element Plus + ECharts │ │  SQLAlchemy 2.0 + SQLite  │
│  Pinia + Vue Router     │ │  JWT + bcrypt 认证       │

## 🏗️ 前端架构

### 技术栈

- **框架**: Vue 3.4+ (Composition API)
- **语言**: TypeScript 5.x
- **UI组件**: Element Plus 2.6+
- **状态管理**: Pinia 2.x
- **路由**: Vue Router 4.x
- **构建工具**: Vite 5.x
- **图表库**: ECharts
- **地图**: Leaflet（支持离线瓦片）

### 核心模块

#### 1. 路由系统

- 使用Vue Router 4实现路由管理
- 支持路由懒加载
- 实现路由守卫进行权限控制
- 使用Hash模式支持离线访问

#### 2. 状态管理

- 使用Pinia进行状态管理
- 模块化设计，每个功能独立store
- 支持状态持久化
- 类型安全的状态访问

#### 3. 数据请求

- 统一的HTTP请求封装
- 请求/响应拦截器
- 错误统一处理
- 支持离线模式

#### 4. 组件系统

- 基于Element Plus构建
- 自定义业务组件
- 组件复用性高
- 响应式设计

## 🔧 后端架构

### 技术栈

- **框架**: FastAPI 0.104
- **语言**: Python 3.11+
- **ORM**: SQLAlchemy 2.0
- **验证**: Pydantic 2.x
- **服务器**: Uvicorn
- **认证**: Python-JOSE + bcrypt
- **数据库**: SQLite
- **缓存**: diskcache（本地磁盘缓存）
- **打包**: PyInstaller

## 📝 技术选型理由

### 为什么选择 Vue 3?

- 性能优秀，响应式系统强大
- Composition API 更灵活
- TypeScript 支持完善
- 生态系统成熟

### 为什么选择 FastAPI?

- 性能出色（基于Starlette）
- 自动生成API文档
- 类型检查（Pydantic）
- 异步支持完善

### 为什么选择 SQLite?

- 零配置，易于部署
- 单文件数据库，便于备份
- 性能满足中小型应用
- 支持完整SQL功能

## 🎯 架构优势

1. **前后端分离**: 开发并行，职责清晰
2. **模块化设计**: 易于维护和扩展
3. **类型安全**: TypeScript + Pydantic
4. **离线支持**: 单机版完全自治
5. **响应式设计**: 适配多种设备
6. **安全可靠**: 多层安全防护
7. **高性能**: 多重优化策略
8. **易部署**: 一键启动脚本

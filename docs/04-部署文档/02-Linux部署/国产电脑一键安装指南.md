# 帮扶管理信息系统 — 国产电脑一键安装指南

> **版本**: v1.2.0 | **更新**: 2026-05-30 | **适用**: 麒麟V10、UOS、Ubuntu 22.04+

---

## 快速开始（3 步安装）

### 步骤 1: 获取 .deb 安装包

将 `assistance-management-system_1.2.0_arm64.deb` 复制到国产电脑任意目录。

### 步骤 2: 一键安装

```bash
sudo dpkg -i assistance-management-system_1.2.0_arm64.deb
```

如果提示缺少依赖，运行：

```bash
sudo apt-get install -f -y
```

### 步骤 3: 访问系统

打开浏览器访问 `http://localhost:8000`

默认账号: `admin`
默认密码: 首次启动自动生成，通过以下命令查看：

```bash
sudo journalctl -u assistance-management-system | grep "密码"
```

## 安装后管理

```bash
# 查看服务状态
sudo systemctl status assistance-management-system

# 启动服务
sudo systemctl start assistance-management-system

# 停止服务
sudo systemctl stop assistance-management-system

# 查看日志
sudo journalctl -u assistance-management-system -f

# 卸载
sudo dpkg -r assistance-management-system
```

## .deb 构建方法

### 方法一：Ubuntu 物理机/虚拟机

在 Ubuntu 22.04+ (x86_64 或 ARM64) 上：

```bash
# 获取源码
git clone <仓库地址>
cd assistance-management-information-system

# 运行构建脚本
chmod +x build-scripts/build-deb-ubuntu.sh
sudo VERSION=1.2.0 ./build-scripts/build-deb-ubuntu.sh
```

产物在 `dist/` 目录下。

### 方法二：Docker 交叉编译（ARM64）

在任意支持 Docker 的机器上：

```bash
# 1. 启用 QEMU 模拟
docker run --rm --privileged multiarch/qemu-user-static --reset -p yes

# 2. 使用 Ubuntu ARM64 容器构建
docker run --rm -v "$(pwd):/build" -w /build \
  --platform linux/arm64 \
  ubuntu:22.04 \
  bash -c "
    apt-get update && apt-get install -y sudo
    cd /build
    sudo VERSION=1.2.0 ./build-scripts/build-deb-ubuntu.sh
  "
```

### 方法三：WSL2 (Windows)

在 Windows 10/11 上启用 WSL2：

```powershell
# 管理员 PowerShell
wsl --install -d Ubuntu-22.04
```

然后在 WSL2 Ubuntu 终端中运行构建脚本。

## 离线安装（无网络环境）

1. 在有网络的机器上准备好 .deb 文件
2. 使用 U 盘复制到目标机器
3. 安装依赖包（预先下载到 U 盘）：

```bash
# 在有网络的机器上
apt-get download libgtk-3-0 libnotify4 libnss3 libxss1 libxtst6 libatspi2.0-0 libuuid1 libgbm1 libasound2 libdrm2
# 将 .deb 文件复制到 U 盘

# 在离线目标机器上
sudo dpkg -i *.deb
```

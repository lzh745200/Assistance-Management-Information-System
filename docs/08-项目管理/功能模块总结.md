## 🔌 API接口文档

### 帮扶村管理API

#### 基本操作
- `GET /api/v1/supported-villages` - 获取帮扶村列表
- `GET /api/v1/supported-villages/{id}` - 获取帮扶村详情
- `POST /api/v1/supported-villages` - 创建帮扶村
- `PUT /api/v1/supported-villages/{id}` - 更新帮扶村
- `DELETE /api/v1/supported-villages/{id}` - 删除帮扶村

#### 年度数据
- `GET /api/v1/supported-villages/{village_id}/yearly-data/{year}` - 获取年度数据
- `POST /api/v1/supported-villages/{village_id}/yearly-data/{year}/population` - 更新人口数据
- `POST /api/v1/supported-villages/{village_id}/yearly-data/{year}/income` - 更新收入数据
- `POST /api/v1/supported-villages/{village_id}/yearly-data/{year}/industry` - 更新产业帮扶数据
- `POST /api/v1/supported-villages/{village_id}/yearly-data/{year}/infrastructure` - 更新基础设施数据
- `POST /api/v1/supported-villages/{village_id}/yearly-data/{year}/party-building` - 更新党建帮扶数据
- `POST /api/v1/supported-villages/{village_id}/yearly-data/{year}/medical` - 更新医疗帮扶数据
- `POST /api/v1/supported-villages/{village_id}/yearly-data/{year}/consumption` - 更新消费帮扶数据
- `POST /api/v1/supported-villages/{village_id}/yearly-data/{year}/employment` - 更新就业帮扶数据
- `POST /api/v1/supported-villages/{village_id}/yearly-data/{year}/education` - 更新教育帮扶数据
- `POST /api/v1/supported-villages/{village_id}/yearly-data/{year}/force-investment` - 更新力量投入数据

#### 模板和导出
- `GET /api/v1/supported-villages/templates/{template_type}` - 下载Excel模板
- `GET /api/v1/supported-villages/export` - 导出帮扶村数据
  - 参数：`year`（年份）、`export_type`（导出类型：basic/comprehensive）

#### 筛选选项
- `GET /api/v1/supported-villages/options/filter` - 获取筛选选项

### 备份管理API

- `GET /api/v1/backup/list` - 获取备份列表
- `POST /api/v1/backup/create` - 创建备份
- `POST /api/v1/backup/restore` - 恢复备份
- `DELETE /api/v1/backup/{backup_id}` - 删除备份
- `GET /api/v1/backup/verify/{backup_id}` - 验证备份
- `GET /api/v1/backup/stats` - 获取备份统计
- `POST /api/v1/backup/cleanup` - 清理过期备份
- `GET /api/v1/backup/download/{backup_id}` - 下载备份

## 📝 使用指南

### 1. 帮扶村管理
1. 登录系统
2. 点击侧边栏"帮扶村管理"
3. 查看帮扶村列表
4. 点击"新增帮扶村"添加新记录
5. 点击"年度数据"管理年度数据
6. 点击"导出数据"导出Excel表格

### 2. 数据导入
1. 点击侧边栏"数据管理" > "数据导入"
2. 选择要导入的数据类型
3. 下载Excel模板
4. 按照模板格式填写数据
5. 上传填写好的文件
6. 查看导入结果

### 3. 经费管理
1. 点击侧边栏"经费管理"
2. 查看经费统计信息
3. 筛选和查询经费记录
4. 新增或编辑经费记录
5. 导出经费报表

### 4. 数据备份
1. 进入主页（工作台）
2. 点击快捷入口"一键备份"
3. 确认备份操作
4. 等待备份完成
5. 或进入"系统管理" > "备份管理"管理备份文件

## 🔧 技术支持

### 常见问题

**Q: 如何重置管理员密码？**
A: 运行 `reset_admin.py` 脚本可以重置管理员密码。

**Q: 数据导入失败怎么办？**
A: 检查Excel文件格式是否符合模板要求，查看错误日志了解具体错误信息。

**Q: 如何备份数据？**
A: 在主页点击"一键备份"按钮，或进入备份管理页面手动备份。

**Q: 系统支持哪些浏览器？**
A: 推荐使用Chrome、Edge、Firefox等现代浏览器。

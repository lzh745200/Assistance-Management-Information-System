## 🚀 部署方式

### 方式一：单机版（推荐）

1. 下载全部文件
2. 双击 `index.html` 或 `启动单机版系统.bat`
3. 使用浏览器打开
4. 输入账号密码登录

**优点:**
- 无需安装
- 完全离线
- 即开即用
- 数据隐私

### 方式二：完整版

1. 安装Python 3.9+
2. 安装依赖: `pip install -r backend/requirements.txt`
3. 启动后端: `python backend/app/main.py`
4. 访问 http://localhost:8000

**优点:**
- 支持并发
- 数据持久化
- API接口
- 可扩展性强

# 主页面（工作台）美化与功能规整 — 设计文档

**日期**: 2026-06-06 | **版本**: 1.0 | **状态**: 已确认

---

## 1. 目标

将 `/dashboard`（HomeSafe.vue，65KB）重构为美观、功能有序的军队乡村振兴工作台首页。

### 1.1 当前问题
- 单文件 65KB，模板/逻辑/样式混杂，维护困难
- 视觉依赖 Element Plus 默认样式，缺乏品牌感
- 功能块罗列无序（布局编辑器、恢复弹窗、动态表单全挤在主页面）
- KPI 卡片仅有数字无趋势，信息密度低

### 1.2 设计原则
- **拆分解耦** — 每个功能块独立为子组件，HomeSafe.vue 仅做布局编排
- **视觉统一** — 复用 Dashboard Step 1-3 的 Token 体系（`.data-number`、militaryTech 主题、`.stat-card`）
- **低频隐藏** — 布局编辑器、恢复数据移至次级入口，主页只保留高频操作
- **功能不变** — 所有现有功能保留，仅调整位置和呈现方式

## 3. 组件拆分

| 组件（新建） | 职责 | 来源 |
|-------------|------|------|
| `views/dashboard/PageHeader.vue` | 欢迎语、日期、3 个高频操作按钮 | 从 HomeSafe 欢迎横幅提取 |
| `views/dashboard/KpiCards.vue` | 5 个核心指标卡片（数量/趋势/sparkline） | 复用 Dashboard Step 2 样式 |
| `views/dashboard/ChartRow.vue` | 双列图表：项目进度 + 经费构成 | 复用 militaryTech ECharts 主题 |
| `views/dashboard/InfoRow.vue` | 左：近期动态时间线，右：快捷入口图标网格 | 从 HomeSafe 拆分 |
| `views/dashboard/index.vue` | 布局编排组件 (~80 行)，替代 HomeSafe.vue | 新建入口 |

**保留但移位的功能**：
- 布局编辑器面板 → 右上角 ··· 下拉菜单 "自定义布局"
- 恢复数据弹窗 → 移至系统管理 / 备份管理页面
- 一键备份按钮 → 保留在 Header 操作区（管理员可见）

## 4. 数据流

```
HomeSafeLayout (index.vue)
  │
  ├─ PageHeader  ← authStore (username, date)
  │
  ├─ KpiCards    ← GET /api/v1/statistics/summary
  │
  ├─ ChartRow    ← GET /api/v1/projects?summary=true
  │                  GET /api/v1/funds/overview
  │
  └─ InfoRow     ← GET /api/v1/activities?limit=10
                     (静态快捷入口配置)
```

所有 API 调用沿用现有端点，不做任何后端改动。

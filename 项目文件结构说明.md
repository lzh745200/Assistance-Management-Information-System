# 帮扶管理信息系统 — 文件结构说明

> **版本**: v1.2.0 | **更新**: 2026-06-12

---

## 📁 backend/app/api/v1/ — API 路由（41 个模块）

| 文件 | 功能 |
|------|------|
| `__init__.py` | 自动扫描加载所有路由模块 |
| `auth/` | 认证：登录、注册、密码重置、Token 管理 |
| `approval.py` | 审批管理：提交申请、批准、拒绝、撤回 |
| `assessment.py` | 成效评估：帮扶效果打分、评分排行 |
| `data_sync.py` | 数据同步：增量导入导出、多机协同 |
| `funds.py` | 经费管理（核心）：CRUD + 状态流转 + 统计 |
| `import_export/` | 异步导入导出 + 报表导出 |
| `map.py` | 地图数据接口：村庄坐标、区域数据 |
| `menus.py` | 菜单权限管理 |
| `messages.py` | 消息通知：系统消息、审批通知 |
| `organization.py` | 组织机构管理 |
| `policy.py` | 帮扶政策：发布、搜索、收藏、导入导出 |
| `projects.py` | 帮扶项目（核心）：CRUD + 状态流转 |
| `school.py` | 帮扶学校管理 |
| `supported_village.py` | 帮扶村（核心）：CRUD + 年度数据 + 帮扶经费 + 导入导出 |
| `system/` | 系统管理：用户、角色、配置、监控、健康检查 |
| `rural_works.py` | 乡村振兴工作记录 |
| `work_logs.py` | 工作日志：操作记录查询 |

> API 完整文档：http://localhost:8000/docs（启动后访问）

---

## 📁 backend/app/core/ — 核心基础设施

| 文件 | 功能 |
|------|------|
| `config.py` | **系统配置**：端口、数据库地址、密钥、版本号（v1.2.0） |
| `security.py` | 认证安全：JWT、密码哈希、Token、bcrypt 兼容 |
| `database.py` | 数据库连接与会话管理 |
| `data_permission.py` | 数据权限：按组织过滤数据访问范围 |
| `cache.py` | 缓存系统：diskcache + 内存 LRU |
| `audit.py` | 审计日志引擎 |
| `events.py` | 事件总线（发布/订阅） |
| `token_manager.py` | Token 黑名单与版本管理 |
| `permission_utils.py` | 权限检查工具 |

---

## 📁 backend/app/models/ — 数据模型（ORM）

| 文件 | 核心模型 |
|------|---------|
| `user.py` | User（用户）、Role（角色） |
| `supported_village.py` | **SupportedVillage**（帮扶村）：布尔字段 `is_revitalization_tier`、`is_tiered_development`；年度子表（人口/收入/产业/基建等 10 个）；帮扶经费 `SupportFunding`；报表订阅 |
| `fund.py` | Fund（经费）：全生命周期字段、状态流转 |
| `project.py` | Project（帮扶项目）：里程碑、进度 |
| `school.py` | School（帮扶学校） |
| `policy.py` | Policy（政策文件） |
| `base.py` | Base、TimestampMixin 基类 |

> Schema 权威来源：模型文件 + Alembic 迁移。已删除：`revitalization_tier`(String)、`tiered_development_level`(String)、`TieredDevelopmentLevel` 查找表。

## 📁 frontend/src/ — 前端核心

### views/ — 页面视图

| 目录 | 主要页面 |
|------|---------|
| `dashboard/` | HomeSafe（工作台）、QuickActions（26 个快捷入口）、自定义布局编辑器 |
| `system/` | MonitoringDashboard（监控面板）、UserManagement、BackupManagement、EncryptionSettings |
| `analytics/supported-villages/` | List（列表+筛选）、Detail（详情+编辑）、SupportedVillageForm（表单）、年度数据 |
| `funds/` | 经费列表、创建/编辑、分析、预算、生命周期 |
| `projects/` | 项目列表、创建/编辑、进度跟踪、导入 |
| `schools/` | 学校列表、详情、数据分析 |
| `dataPackage/` | 数据包版本、增量更新、任务管理 |
| `dataEntry/` | 综合数据录入 |
| `auth/` | 登录、注册、忘记密码、修改密码 |

### 其他目录

| 目录 | 内容 |
|------|------|
| `components/` | 通用组件（图表、表格、地图选择器）+ 业务组件（VillageCard 等） |
| `stores/` | Pinia 状态：auth、menu、config、app |
| `router/` | 路由配置、权限守卫 |
| `api/` | API 请求封装（request + 各模块 API） |
| `types/` | TypeScript 类型定义（analytics、helpProject、api 等） |
| `config/` | 菜单配置、路由映射 |
| `utils/` | 工具函数：enhancedStorage、logger、pagination |
